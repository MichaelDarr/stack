# Frontend

## Directory Structure

| Directory      | Contents                               |
| :------------- | :------------------------------------- |
| `src`          | Source Code                            |
| `src/@types`   | Global TypeScript Type Definitions     |
| `src/contexts` | Application-Level React Contexts       |
| `src/lib`      | Shared application code                |
| `src/lib/react`| React-specific shared application code |
